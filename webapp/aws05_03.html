AWS WAF, AWS WAF Classic에서 WAF 규칙을 변환하는 마이그레이션 마법사 지원

Data : Apr 29, 2020

이제 AWS WAF가 AWS WAF Classic에서 WAF 규칙을 마이그레이션할 수 있는 간단한 방법을 제공합니다. 마이그레이션 마법사는 웹 ACL을 통해 구문 분석을 수행하고 S3 버킷으로 CloudFormation 템플릿을 생성합니다. 해당 CloudFormation 템플릿을 배포하여 새 AWS WAF에 해당하는 웹 ACL을 생성할 수 있습니다.

마이그레이션 마법사를 사용하면 AWS WAF Classic에서 기존 웹 ACL을 새 AWS WAF로 변환할 수 있으므로 WAF 규칙을 수동으로 변환하는 시간과 노력을 절감할 수 있습니다.

마이그레이션 마법사는 AWS WAF가 오늘 정식 출시되는 모든 AWS 리전에서 사용할 수 있습니다. 마이그레이션 마법사 사용에 대한 추가 비용은 없습니다(CloudFormation 템플릿에서 S3 요금을 뺀 금액).

AWS WAF Classic 웹 ACL 마이그레이션을 시작하려면 AWS WAF Classic 콘솔로 이동하여 마이그레이션 프로세스를 시작하십시오.

콘솔 마법사를 시작했으면 마이그레이션하려는 웹 ACL과 템플릿을 저장할 S3 버킷을 지정하십시오. 그런 다음 AWS CloudFormation 콘솔로 이동하여 해당 템플릿에서 스택을 생성한 다음 새 AWS WAF에서 웹 ACL을 다시 생성할 수 있습니다. 마이그레이션에 대한 자세한 내용은 AWS WAF 개발자 안내서를 참조하십시오.
