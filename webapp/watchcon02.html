솔루션 아키텍쳐

Watchcon Solution
클라우드 환경에 최적화 된 시스템 관리 및 서버 모니터링을 위한 맞춤형 솔루션입니다.


Watchcon Solution 아키텍처는 정보 수집을 담당하는 Agent와 수집된 정보를 가공 처리 하는 서버로 구성되어 있습니다.
수집 되는 정보의 구성은 데이터 베이스가 운영되는 OS 정보를 포함하여 Instance / Session / SQL 레벨 등의 정보로 구성되어 있으며, 운용자는 Watchcon Solution 웹서버에 접속하여 다양한 형태(관제/모니터링/분석/튜닝/리포트)의 멀티 인스턴스 통합 감시를 수행할 수 있습니다.

Watchcon Architecture



클라우드 환경에 최적화 된 맞춤형 모니터링 솔루션에 대해 더 자세히 알고 싶다면, 웅진입니다.
